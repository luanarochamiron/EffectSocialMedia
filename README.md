# Neumorphism Effect on Social Media Icons - CodingLab

Um componente de interface de usuário moderno, elegante e totalmente responsivo que apresenta botões de redes sociais estilizados com a estética do Neumorfismo (Neumorphism / Soft UI). Desenvolvido puramente com HTML5 e CSS3, o projeto destaca a aplicação de múltiplas sombras em direções opostas para criar elementos circulares táteis que parecem moldados diretamente sobre a superfície do plano de fundo.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica da lista e dos links de direcionamento social (`<a>`).
* CSS3: Estilização completa, uso de bordas circulares, paleta monocromática suave e gerenciamento de sombras projetadas avançadas.
* Flexbox: Distribuição horizontal perfeitamente simétrica entre os ícones e centralização absoluta de todo o bloco na viewport.

## Funcionalidades e Técnicas Aplicadas

* Estética Soft UI Realista: Cada botão circular utiliza uma combinação precisa de sombras claras (brancas) no canto superior esquerdo e sombras escuras (cinzas) no canto inferior direito, simulando uma iluminação natural em relevo.
* Lista de Plataformas Populares: Inclusão de marcas e redes de alto engajamento: Facebook, Twitter, Instagram, GitHub e YouTube, renderizadas com suas respectivas cores de identidade visual.
* Microinterações no Hover (Animação Neumórfica): Transição suave que altera o comportamento das sombras ao passar o mouse, criando um efeito de clique ou afundamento visual (*inset shadow*) altamente interativo.
* Layout Minimalista e Limpo: Uso de uma paleta de cores monocromática em tons de cinza claro fosco para garantir a suavidade visual e eliminar divisores agressivos na interface.
